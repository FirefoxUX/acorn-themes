# Acorn Themes
Firefox themes based off of Acorn's [color palette](https://acorn.firefox.com/latest/desktop/styles/color/semantic-color-14S4X1uo)

## Formats Available

- **OKLCH themes** (root folders): Original themes using OKLCH color format
- **RGB arrays** (`rgb-arrays/` folder): Same themes converted to RGB arrays for broader compatibility

Both formats provide identical visual appearance.
